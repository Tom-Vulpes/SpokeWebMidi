⚙️ Setup Instructions
1. Install CircuitPython:
  1. Download the CircuitPython .uf2 file for Raspberry Pi Pico
  2. Hold the BOOTSEL button on your Pico while plugging in USB
  3. Drag the .uf2 file to the RPI-RP2 drive that appears
  4. Board will reboot and appear as CIRCUITPY drive
2. Install Libraries:
  1. Download the CircuitPython Library Bundle
  2. Extract the bundle ZIP file
  3. Copy these folders to CIRCUITPY/lib/:
    • adafruit_midi/
    • adafruit_hid/
3. Upload Code:
  1. Click "Download code.py" button above
  2. Copy downloaded file to CIRCUITPY drive
  3. Board will auto-restart and run your code!
